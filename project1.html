Project Overview

Time Travel Adventure is a narrative-driven game that explores different historical eras through time travel. Developed in Unity, it features dynamic environments, character animations, and complex time-based puzzles. The game also utilizes custom shaders for unique visual effects.

Key Features:

  • Time travel mechanics with interactive timelines
  • Detailed 3D environments across multiple eras
  • Advanced physics and lighting effects
  • Immersive story with branching narrative paths

Technologies Used:

  • Unity
  • C# Programming
  • Custom Shaders
  • AI Pathfinding
  • Physics Simulations
