Project Overview

Time Travel Adventure is an immersive narrative-driven game where players experience different historical periods through time travel. The game was built using Unity, and it features dynamic environments, a detailed story, and complex gameplay mechanics involving time-based puzzles.

As the protagonist, you will explore ancient civilizations, future dystopias, and alternate versions of the world, solving puzzles to manipulate time and prevent a catastrophic event. The game showcases advanced physics, shaders, and a unique time travel mechanic that impacts the gameplay and narrative.

Key Features:

  • Time travel mechanic that allows players to jump between multiple time periods.
  • Rich, detailed environments, each with its own history and narrative.
  • Complex AI behavior and puzzle-solving that alters the gameâ€™s world dynamically.
  • Original soundtrack and sound design, adding depth to the atmosphere of each time period.
  • Branching narrative paths based on player choices, leading to multiple possible endings.

Technologies Used:

  • Unity - Primary game engine for development.
  • C# - Programming language for game mechanics and systems.
  • Custom Shaders - For time-based effects and visual transitions between time periods.
  • AI Pathfinding - Using Unity's NavMesh for enemy AI behavior.
  • Physics Simulations - Advanced physics systems for interacting with time and environment.

Challenges & Solutions

During the development of Time Travel Adventure, I faced several challenges, particularly with the time travel mechanic. Ensuring smooth transitions between different time periods while maintaining a consistent player experience was difficult. To overcome this, I developed custom shaders that allowed seamless transitions between worlds, and implemented a time-based event system that dynamically altered the environment.
